vscode 插件将 [unocss](https://github.com/unocss/unocss) 转换为 css。它由 @unocss/core 提供支持。

![demo](/assets/demo.gif)

## Config
```
// 自定义配置高亮样式
// settings.json
  "unocss-to-css":{
    "dark": {
      "textDecoration": "underline",
      "backgroundColor": "rgba(144, 238, 144, 0.5)",
      "color": "black",
    },
    "light":{
      "textDecoration": "underline",
      "backgroundColor": "rgba(255, 165, 0, 0.5)",
      "color": "#ffffff",
    }
  }
```
